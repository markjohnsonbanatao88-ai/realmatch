# Security and Privacy Notes

Current status: static MVP only.

## Current posture

- No backend is active.
- No authentication is active.
- No database is active.
- No payment integration is active.
- No identity-document upload is active.
- Application form submission is local UI state only.

## Required hardening before live data

- Role-based admin authentication.
- Server-side validation.
- Secure database rules.
- Audit logging.
- Consent records.
- Retention and deletion rules.
- Encrypted storage for any sensitive material.
- Staff access controls.
- Incident response process.
- Backups and recovery plan.

## Hard rule

Do not collect real applicant data until privacy, security, retention, and access-control workflows are implemented and reviewed.
