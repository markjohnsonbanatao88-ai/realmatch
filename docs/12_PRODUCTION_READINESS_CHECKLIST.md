# Production Readiness Checklist

Current status: static MVP only. This checklist is a release gate, not proof of readiness.

## Static MVP checks

- [ ] Homepage explains the product boundary clearly.
- [ ] Pricing page avoids outcome promises.
- [ ] Application flow says no live backend exists yet.
- [ ] Footer and conversion pages include the legal disclaimer.
- [ ] Terms and privacy pages are marked as placeholders until counsel review.
- [ ] Admin dashboard uses mock data only.
- [ ] CI runs install, lint, typecheck, and build.

## Pre-launch checks

- [ ] Terms reviewed by counsel.
- [ ] Privacy notice reviewed by counsel.
- [ ] Consent records implemented.
- [ ] Opt-out workflow implemented.
- [ ] Secure storage rules implemented.
- [ ] Role-based admin access implemented.
- [ ] Audit logs implemented.
- [ ] Complaint workflow implemented.
- [ ] Refund policy finalized.
- [ ] Staff playbook finalized.

## No-go conditions

Do not launch if the service is being sold as guaranteed outcomes, staff are improvising policy, or real personal data is collected without security and retention controls.
